📖 How to View Diagrams

Option 1: Online PlantUML Renderer

Visit plantuml.com/plantuml and paste the .puml file content

Option 2: VSCode Extension

Install the "PlantUML" extension and open any .puml file to see live preview

Option 3: Generate Locally

# Install PlantUML brew install plantuml # Generate all diagrams cd docs/architecture plantuml *.puml -png

Option 4: Docker

docker run --rm -v $(pwd):/data plantuml/plantuml -png /data/*.puml
